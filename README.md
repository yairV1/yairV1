No empecé con un bootcamp. Empecé con curiosidad.

Soy desarrollador en formación, pero no del tipo que espera que alguien le explique — soy del tipo que abre la documentación, rompe el código, entiende el error y lo resuelve. Mi stack no define quién soy; lo que construyo con él, sí.
